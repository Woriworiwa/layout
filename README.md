# Layout - Low-Code CSS Layout Generator

A visual canvas-based editor for creating responsive layouts using CSS Grid and Flexbox.
Built with modern Angular, this tool simplifies the process of creating complex layouts through an intuitive drag-and-drop interface that generates clean CSS, HTML, and JSON output.

**Demo**: https://demo.dalgamuni.me
